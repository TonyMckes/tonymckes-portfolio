# Personal portfolio

Build with Typescript, Next.js and CSS Modules. 

## Install
First, you need to install the depencencies

```sh
npm install
```

## Usage

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
